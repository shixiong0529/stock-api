# Stock API Plan Archive

这是 `stock-api` 项目的历史计划稿归档版本。

为避免在公开仓库中暴露本机用户名和目录结构，本文件中的路径统一改为占位写法：

- `<OPENCLAW_HOME>`
  - 例如 `C:\Users\<YOUR_USER>\.openclaw`

- `<WORKSPACE_ROOT>`
  - 例如 `<OPENCLAW_HOME>\workspace-stock`

- `<PROJECT_ROOT>`
  - 例如 `<WORKSPACE_ROOT>\stock-api`

原计划的核心目标：

- 在 `<PROJECT_ROOT>` 下实现本地 FastAPI 行情服务
- 封装 A 股、港股个股接口
- 提供市场总览接口
- 与 OpenClaw `stock` agent 集成

原计划涉及的修改范围：

- `<PROJECT_ROOT>`
- `<WORKSPACE_ROOT>\TOOLS.md`
- `<WORKSPACE_ROOT>\AGENTS.md`
- `<OPENCLAW_HOME>\openclaw.json`

当前这份归档文件仅用于保留计划背景，不再作为运行文档。

如需查看实际实现，请以这些文件为准：

- `README.md`
- `PROJECT_SUMMARY.md`
- `INTERFACES.md`
- `main.py`
- `sources.py`
