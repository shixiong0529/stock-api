# Stock API Prompt Archive

这是 `stock-api` 项目的历史执行提示词归档版本。

为避免暴露本机绝对路径，本文件中的路径统一使用占位符：

- `<OPENCLAW_HOME>`
- `<WORKSPACE_ROOT>`
- `<PROJECT_ROOT>`

该提示词当时约束的核心内容包括：

- 读取项目计划文件后逐步执行
- 以 `stock-api` 为目标目录实现本地行情服务
- 修改工作区规则文件与 agent 文档
- 保持测试、验证和进度更新

当前保留本文件，仅用于项目留档。
